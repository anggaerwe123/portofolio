Nama Saya Angga Rizki Wijaya

Saya adalah seorang siswa SMK PGRI 3 Malang di bidang rekayasa perangkat lunak, khususnya dalam pengembangan aplikasi web. Saya memiliki pengalaman dalam membangun aplikasi berbasis PHP, MySQL, HTML, CSS, Bootstrap, serta JavaScript, dengan kemampuan kuat dalam mengelola database, membuat fitur CRUD, dan mengimplementasikan session handling. Selain itu, saya juga familiar dengan framework Laravel untuk pengembangan web. Saya terbiasa bekerja dengan Visual Studio Code, XAMPP, dan selalu berusaha untuk menghasilkan solusi yang efektif dan user-friendly. Dengan pengetahuan yang terus berkembang, saya siap untuk berkontribusi dalam praktik kerja lapangan di perusahaan Autochem.

Saya ingin memanfaatkan keterampilan dan pengetahuan yang telah saya peroleh dalam pengembangan aplikasi web untuk memberikan kontribusi nyata dalam tim pengembangan perusahaan. Dengan mengikuti praktik kerja lapangan, saya berharap dapat memperdalam pemahaman saya dalam penerapan teknologi terbaru, sambil mengembangkan solusi inovatif yang mendukung efisiensi dan produktivitas perusahaan. Saya berambisi untuk terus belajar dan berkembang dalam bidang ini, serta berkontribusi dalam proyek yang berdampak pada perkembangan perusahaan di masa depan.
